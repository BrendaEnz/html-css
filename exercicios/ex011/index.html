Usando link externo

É um link que acessa um site que não é nosso

Você pode acessar o meu repositorio publico no GitHub

Você tambem pode acessar o meu canal gratuito no youtube

Para deixar o link abrir em uma nova pagina é necessario adicionar dentro ta tag de link a seguinte tag /target="_blank" rel="externo"
